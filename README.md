# 🍽️ Sistema Restaurante Digital

Sistema completo para gerenciamento operacional de restaurantes, bares, lanchonetes, quiosques e estabelecimentos do setor alimentício.

Desenvolvido para centralizar o atendimento, gerenciamento de comandas, delivery, controle de produtos, clientes e funcionários, oferecendo uma operação mais eficiente e organizada.

---

## 📜 Notas

* Status: Em Desenvolvimento
* Versão: 1.1.0
* Projeto desenvolvido paralelamente ao Projeto Legion
* Plataforma Web Responsiva
* Compatível com Desktop, Tablet e Smartphones

## ⚙️ Instalação

### Clonar o repositório

```bash
git clone https://github.com/seu-usuario/comandas.git
```

### Entrar na pasta do projeto

```bash
cd comandas
```

### Instalar dependências

```bash
npm install
```

---

## ▶️ Executando Localmente

```bash
npm start
```

A aplicação será iniciada em:

```text
http://localhost:3000
```

---

## 🏗️ Gerando Build de Produção

```bash
npm run build
```

Os arquivos serão gerados na pasta:

```text
/build
```
